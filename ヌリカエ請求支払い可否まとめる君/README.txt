使用にあたっての下準備は以下の通り
使用時は使用するPCにPythonもしくはAnaccondaをインストールしてください。
インストールしたらPython単体の場合、プロンプトからpip install ライブラリ名　
Anacondaの場合、conda install ライブラリ名　で以下のライブラリをインストールしてください

re
os
csv
glob
time
openpyxl
pandas
datetime
bs4
selenium 
pywin32

ブラウザの操作の為、ドライバをインストールする必要があります
推奨ブラウザはGoogle Chromeです
ブラウザのバージョンとドライバのバージョンを合わせておき、インストールしたドライバのフルパスを本文22行目に書いておきます

Anacondaフォルダ下のpythonw.exeで常にpythonプログラムを開くように設定して、ダブルクリックでプログラムを起動させるようにしてください。
