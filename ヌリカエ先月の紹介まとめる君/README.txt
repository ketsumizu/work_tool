nurikae_this_month.pyは
ヌリカエから（プログラム実行時から見て）先月紹介された案件の一覧をまとめて、
Excelデータにしてくれるpythonプログラムです

完成した表は社内連絡用として、営業管理と経理部長へ１セットずつ毎月渡すものです

以下のライブラリを使用しているので初めて使うPCの場合pipまたはcondaでインストールしてください
chrome用のドライバも必要なのでこれもインストールして、コード内のパスをインストール先の絶対パスに修正

os
csv
glob
time
calendar
openpyxl
datetime
selenium
win32com.client  


プログラムの最後のほうに、作ったExcelの表をpdfの形式で吐き出すためのコードがありますが、コメントアウトしてあります
必要であればコメント解除して使ってください


なんかうまく動かないなって時はまず、コード内のパスの指定あたりがうまくいってないと思われます
確認して修正してください
